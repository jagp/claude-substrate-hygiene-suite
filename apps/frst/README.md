# apps/frst/

FRST64 scan output and fix artifacts.

- `archive/[yyyy-mm-dd]/` — `FRST.txt`, `Addition.txt`, `Shortcut.txt` (gitignored)
- `fixlists/[yyyy-mm-dd]-FIXLIST.txt` — generated FIXLISTs (gitignored)
- `pending/` — FIXLISTs staged for review before deployment (gitignored)
