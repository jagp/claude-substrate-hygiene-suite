# FRST/

Scan output and fix artifacts for FRST64 (Farbar Recovery Scan Tool).

- `archive/[YYYY-MM-DD]/` — raw scan output: `FRST.txt`, `Addition.txt`, `Shortcut.txt` (gitignored)
- `fixlists/[YYYY-MM-DD]-FIXLIST.txt` — generated FIXLISTs (gitignored)
- `pending/` — FIXLISTs staged for review before deployment (gitignored)

See [`tools.md`](../tools.md) for invocation details.  
See [`procedure.md`](../procedure.md) for the full FRST workflow (Phases 1–3).
